# Dudi - Project Overview & Interactive Mockups

במסגרת הפרויקט פותחה מערכת המשלבת ניהול תהליכים עסקיים עם דגש על חוויית משתמש ופונקציונליות טכנולוגית.

## 🚀 Live Demo
ניתן לצפות במוקאפים האינטראקטיביים ובממשק המשתמש בקישור הבא:
[צפייה בממשק הפרויקט](https://newmagal78-hash.github.io/Dudi-mockupd/)

## 📂 תכולת הפרויקט
ה-Repository הזה מרכז את כל נכסי הפרויקט בשלב האפיון והעיצוב:

* **index.html**: נקודת הכניסה להצגת המוקאפים בדפדפן.
* **PRD.md**: מסמך דרישות מוצר (Product Requirements Document).
* **STATUS.md**: סטטוס התקדמות עדכני של הפרויקט.
* **STATUS_FLOW.md**: תיאור זרימת התהליכים במערכת.
* **mockups/**: תיקייה המכילה את כל הנכסים הוויזואליים והעיצובים.

## 🛠️ טכנולוגיות בשימוש
* HTML5 / CSS3 לטובת בניית ה-Prototype.
* Markdown לניהול תיעוד טכני ועסקי.
* GitHub Pages לאירוח (Hosting) של הממשק.
